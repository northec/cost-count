# 使用说明

这份说明专为直接使用 EXE 程序的用户准备，操作简单易上手，两种使用方式可按需选择~

## 二、使用方式（任选其一即可）

### 方式一：直连法——放入目标文件夹运行

这是最便捷的方式！只需将 `file_counter_YYYY.MM.DD.N.exe` 文件，直接复制或移动到你需要统计文件的文件夹中，双击启动程序就好。

程序会自动扫描当前文件夹及所有子文件夹的内容，无需额外操作，最终生成 Excel 统计报告。

### 方式二：灵活适配法——任意位置运行并指定目录

如果不想移动 EXE 文件，可选择这种方式，支持两种指定路径的操作：

1. 图形化提示输入：双击 EXE 后，界面会弹出路径输入提示，直接粘贴目标文件夹的路径并按回车，程序就会开始统计；若直接回车（不输入内容），则默认统计 EXE 所在的当前文件夹。

2. 命令行指定：适合习惯用命令行操作的用户，输入以下命令即可（替换路径为你的实际文件夹路径）：

示例：

```
file_counter_2026.01.30.1.exe "D:\path\to\directory"
```

## 三、输出说明

统计完成后，程序会在你指定的目标目录下，生成一份带唯一时间戳的 Excel 报告，示例文件名如下：

文件统计报告_20260130_140632.xlsx

为了方便你查看结果，程序默认会在报告生成后自动打开该 Excel 文件。

小提示：时间戳由年月日时分秒组成，可避免多次统计的报告覆盖，方便你留存不同时间的统计结果。
