# EXE 使用说明

本说明面向直接使用已打包 EXE 的场景。

## 文件位置

EXE 默认在 `dist/` 目录，命名规则：

```
file_counter_YYYY.MM.DD.N.exe
```

## 使用方式

1) 双击运行  
不带参数时会提示输入目录，留空统计当前目录。

2) 命令行运行  

```bash
dist/file_counter_2026.01.30.1.exe "D:\path\to\directory"
```

## 关闭自动打开

如不希望自动打开生成的 Excel，执行前设置：

```bash
set FILE_COUNTER_NO_OPEN=1
```

## 输出

输出文件示例：

```
文件统计报告_20260130_140632.xlsx
```
